# 🌳 Arboris: Tree Visualization Lab

Arboris is a high-fidelity, interactive platform designed to demystify data structures through cinematic visualizations. Built with a focus on educational precision and premium aesthetics, it allows students and engineers to master the flow of complex recursive logic.

## ✨ Core Features

### 🔹 1. Multi-Model Support
- **Binary Tree (BT)**: Observe basic structural depth and level-order transitions.
- **Binary Search Tree (BST)**: Master recursive search, insertion, and deletion logic.
- **AVL Tree**: Experience automatic structural stability through high-fidelity rotation animations.

### 🔹 2. Cinematic Step-Logic
Every operation (Insert, Search, Delete) is accompanied by a **Step Logic Stream** in the interactive terminal.
- **Visual Tracers**: Nodes glow as the algorithm "visits" them.
- **Educational Context**: Real-time explanations of why the algorithm chooses a specific path.

### 🔹 3. AVL Balance Architecture
- **Automatic Rotations**: Watch Right, Left, Left-Right, and Right-Left rotations happen in real-time.
- **Live Factors**: Every node displays its height and balance factor ($H_L - H_R$) dynamically.

### 🔹 4. Logic Mirror
- Integrated **Monaco Editor** view to compare the visual visualization with real C++ implementation code.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/arboris-tree-lab.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Launch the laboratory:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack
- **Framework**: React 19 (Modern Hooks-centric)
- **Styling**: Vanilla CSS + Tailwind CSS 4.0 for glassmorphic effects.
- **Icons**: Lucide React.
- **Editor**: @monaco-editor/react.
- **Build Tool**: Vite 8.

---

## 🎨 Design Philosophy
Arboris uses a **Submarine-Dark** aesthetic with:
- **Glassmorphism**: Translucent panels and vibrant blurs.
- **Micro-animations**: Smooth SVG transitions for node positioning and connection drawing.
- **Terminal Logic**: A dedicated sidebar for deep algorithmic understanding.
